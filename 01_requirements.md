# 間取り作成ツール「マドリエ」 要件定義書

## 1. プロジェクト概要

### 1.1 目的

注文住宅の集客を目的とした、ブラウザベースの無料間取り作成ツールを開発する。
ユーザーが直感的に理想の間取りを作成できる体験を提供し、「この間取りでいくらになるか」という見積もり相談・建築相談への導線を構築する。

### 1.2 ビジネスゴール

- 注文住宅に興味のある潜在顧客に無料ツールとしてリーチする
- 間取り作成→見積もり相談という自然な導線で問い合わせ数を増加させる
- 作成された間取りデータを営業資料として活用する

### 1.3 ターゲットユーザー

- 注文住宅を検討中の一般消費者（30〜50代が中心）
- 建築・設計の専門知識を持たない人
- スマートフォン・タブレット・PCいずれからもアクセスする可能性がある

### 1.4 スコープ外（本ツールでは対応しない）

- 建築基準法等の法令チェック
- 構造計算・耐震計算
- 正確な建築費見積もりの自動算出
- 3Dパース・ウォークスルー（将来フェーズで検討）

## 2. 機能要件

### 2.1 間取り作成機能（コア機能）

#### 2.1.1 テンプレート選択（スタート方式）

| 項目 | 内容 |
|------|------|
| 概要 | 定番の間取りテンプレートから開始できる |
| テンプレート例 | 2LDK（25坪）、3LDK（30坪）、4LDK（35坪）、3LDK+S（32坪）など |
| 操作 | テンプレート一覧から選択 → 編集画面に遷移 |
| 白紙スタート | テンプレートを使わず、白紙から自由に作成も可能 |

#### 2.1.2 部屋パーツの配置・編集

| 項目 | 内容 |
|------|------|
| パーツ一覧 | サイドバーに部屋パーツをカテゴリ分けして表示 |
| 配置操作 | パーツをドラッグ&ドロップでキャンバスに配置 |
| グリッドスナップ | 半畳（910mm×910mm）単位でスナップ |
| サイズ変更 | 部屋の四辺・角をドラッグしてリサイズ（グリッド単位） |
| 回転 | 90度単位で回転可能 |
| 削除 | 選択→Deleteキーまたは削除ボタン |
| 複製 | 選択→複製ボタン |

#### 2.1.3 部屋パーツの種類

**居室系**
- LDK（リビングダイニングキッチン）
- リビング
- ダイニング
- キッチン
- 主寝室
- 洋室
- 和室
- 子供部屋
- 書斎・ワークスペース

**水回り系**
- 浴室（ユニットバス）
- 洗面脱衣室
- トイレ
- ランドリールーム

**収納系**
- ウォークインクローゼット
- シューズクローク
- パントリー
- 納戸

**その他**
- 玄関
- 廊下
- 階段
- バルコニー・ベランダ
- ウッドデッキ・テラス
- ガレージ

#### 2.1.4 建具の配置

| 項目 | 内容 |
|------|------|
| ドア | 開き戸（開き方向指定可）、引き戸 |
| 窓 | 引き違い窓、FIX窓、出窓 |
| 配置方法 | 部屋の壁上にスナップして配置 |

#### 2.1.5 階数対応

| 項目 | 内容 |
|------|------|
| 対応階数 | 1階・2階の切り替え表示 |
| 階段 | 1階に階段パーツを配置すると2階にも連動表示 |

### 2.2 面積自動計算

| 項目 | 内容 |
|------|------|
| 各部屋面積 | 配置された各部屋の㎡数・畳数を自動計算して表示 |
| 延床面積 | 全部屋の合計面積を自動計算 |
| 坪数表示 | 延床面積を坪数に変換して表示（1坪=3.31㎡） |
| 階別面積 | 1階・2階それぞれの面積を表示 |

### 2.3 表示・操作機能

| 項目 | 内容 |
|------|------|
| ズーム | ピンチ操作・ホイール・ボタンで拡大縮小 |
| パン | ドラッグで表示領域を移動 |
| Undo/Redo | 操作の取り消し・やり直し |
| グリッド表示 | ON/OFF切り替え可能 |
| 寸法線表示 | 部屋サイズをmm単位で表示 |

### 2.4 保存・出力機能

| 項目 | 内容 |
|------|------|
| ブラウザ保存 | ローカルストレージに自動保存（※注） |
| 画像出力 | PNG形式で間取り図をダウンロード |
| PDF出力 | A4サイズのPDFとして出力（将来対応可） |
| 共有リンク | URLパラメータ等で間取りデータをエンコードして共有（将来対応可） |

※注: Reactアーティファクトとして動作させる場合はlocalStorage不可。React stateまたはpersistent storageで代替する。

### 2.5 集客導線機能

| 項目 | 内容 |
|------|------|
| 見積もり相談ボタン | 画面上に常時表示。タップで問い合わせフォームへ遷移 |
| 問い合わせフォーム | 名前・電話番号・メールアドレス・希望エリア・ご要望（自由記述） |
| 間取りデータ送信 | フォーム送信時に作成した間取りデータ（JSON + 画像）を添付 |
| CTA表示 | 適切なタイミングでバナーやモーダルで「プロに相談しませんか？」を表示 |
| 会社情報 | フッター等に藤島建設の会社情報・施工事例リンクを掲載 |

## 3. 非機能要件

### 3.1 パフォーマンス

- 初期ロード: 3秒以内（3G回線でも5秒以内）
- 操作レスポンス: ドラッグ&ドロップ60fps維持
- 部屋パーツ50個以上でもスムーズに動作

### 3.2 対応デバイス・ブラウザ

| デバイス | ブラウザ | 対応レベル |
|---------|---------|-----------|
| PC | Chrome / Edge / Safari / Firefox（最新2バージョン） | フル対応 |
| タブレット | iPad Safari / Chrome | フル対応（タッチ操作） |
| スマートフォン | iOS Safari / Android Chrome | 基本対応（簡易操作モード検討） |

### 3.3 アクセシビリティ

- キーボード操作対応（Tab移動、Enter選択、Delete削除）
- 十分なコントラスト比の確保
- ツールチップによる操作ガイド

### 3.4 SEO・OGP

- 適切なmeta情報の設定
- OGP画像の設定（SNS共有時のプレビュー）
- 「間取り 作成 無料」等のキーワードでの検索流入を意識

## 5. ユーザーフロー

```
[トップページ]
    ├─ テンプレートを選択 ──→ [間取り編集画面（テンプレート読み込み済み）]
    └─ 白紙から作成 ────────→ [間取り編集画面（空のキャンバス）]
                                     │
                                     ├─ 部屋パーツを追加・配置・リサイズ
                                     ├─ 建具（ドア・窓）を配置
                                     ├─ 面積・坪数を確認
                                     ├─ 画像として保存
                                     │
                                     └─ 「この間取りで相談する」ボタン
                                          │
                                          ▼
                                   [問い合わせフォーム]
                                   （間取りデータが自動添付）
                                          │
                                          ▼
                                     [送信完了画面]
```

## 6. 開発フェーズ

### Phase 1: MVP（最小実用プロダクト）

- テンプレート選択（3種類程度）
- 部屋パーツの配置・移動・リサイズ・削除
- 面積自動計算
- 画像出力（PNG）
- 見積もり相談ボタン（外部フォームへのリンク）
- PC対応のみ

### Phase 2: 機能拡充

- テンプレート追加（6〜10種類）
- 建具（ドア・窓）の配置
- 2階対応
- Undo/Redo
- タブレット・スマホ対応
- 問い合わせフォーム内蔵 + 間取りデータ送信

### Phase 3: 集客強化

- PDF出力
- 共有リンク機能
- CTA最適化（タイミング・文言のA/Bテスト）
- SEO対策（ブログ記事連携等）
- アクセス解析連携（GA4）

### Phase 4: 将来構想

- 3Dビュー / ウォークスルー
- 家具・インテリア配置
- AI間取り提案（条件入力→自動生成）
- 概算見積もり自動表示
- ユーザーアカウント・間取り保存機能

## 7. 用語定義

| 用語 | 定義 |
|------|------|
| グリッド | キャンバス上のスナップ単位。半畳（910mm×910mm）を基本単位とする |
| パーツ | 部屋・建具などの配置可能なオブジェクト |
| テンプレート | 事前に用意された間取りのプリセット |
| 坪 | 面積単位。1坪 = 約3.31㎡ = 2畳 |
| 畳 | 面積単位。1畳 = 約1.62㎡（中京間基準） |
| CTA | Call To Action。ユーザーに行動を促すUI要素 |
